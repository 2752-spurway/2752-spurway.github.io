# 2752 S Spurway Dr — Listing Site

A single-page listing site for a townhome in Ann Arbor, MI. Static and
self-contained — open `index.html` or deploy the folder as-is.

## Sections
- **Gallery** — photo carousel
- **Floor plan** — the builder's drawings with every connected device mapped; hover a dot for a product card, tap a legend chip to light up a category
- **Smart home** — Lutron Caséta, Level Lock+, ecobee, Aqara, Genie, IKEA
- **Water** — whole-home softener + reverse-osmosis drinking water, with a hardness figure

## Deploy (GitHub Pages)
For a `<name>.github.io` repo, Pages publishes automatically from the default
branch. Otherwise: Settings → Pages → Source `main` / `/ (root)`. `.nojekyll`
is present so all files are served verbatim.

## Contents
- `index.html` — the whole page (styles + scripts inline)
- `img/` — floor plans, gallery, product shots, brand logos
